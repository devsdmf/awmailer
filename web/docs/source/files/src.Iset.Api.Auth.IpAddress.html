M4A1 - The Awesome Mailer Service

The M4A1 is a software developed for provide a mail service which can be used by all services of iSET.

The proposal of M4A1 is provide a mail tool that runs a daemon as a observer for new services to be triggered, this services runs natively on Linux servers independent of each others.

This is a source code file, part of M4A1 product and this source code is privately and only iSET and your developers can use or distribute it.
